# COMPUTOOLS PROJECT

## Requirements
1. NodeJS: Recommended version - v14.17.3

2. Typescript: Recommended version - 4.5.5

3. Install all the required node modules:
    >npm install

## Build
To build the app:
>tsc

To build the docker image:
>docker-compose build

## Commands
To run the project:
>npm run start 

To run the project in the development mode:
>npm run start:dev

Migrations up:
>npm run migration:up

Migrations down:
>npm run migration:down

Seeds up:
>npm run seed:up

Seeds down:
>npm run seed:down

To reset the test DB:
>npm run reset-tdb

To start testing the app:
>npm run test

Docker up:
>docker-compose up

Docker down:
>docker-compose down
